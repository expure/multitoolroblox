# 🚀 Character Controller & Utilities Module

Enhance your movement, control, and interaction in-game. A powerful set of tools for advanced players.

---

## 📖 Description

This module provides enhanced character control capabilities, including multiple speed modes, movement stabilization, aiming, and physical interactions.

---

## ⚡ Recommended Speed Method

> **✅ Legit** — the main mode for realistic and safe speed boosting.

- Use **Strength Mode** to push/pull heavy objects  
- If **Grabber** or **Puncher** don't work:  
  Try to **nudge**, **jump**, or **touch** the target — this will activate them

---

## 🎯 Stabilization & Control

| Feature          | Purpose |
|----------------|-------------|
| **Y‑Stab** | Counteracts unintended upward impulses (e.g., from high speed), keeping you grounded while running |
| **Move‑Stab** | Prevents rubberbanding or movement without your input |

---

## 🖱️ AimBot

> Hold **RMB (Right Mouse Button)** — camera instantly snaps to the nearest player.

---

## ⚙️ Features (excluding Grabber/Puncher)

- **FullBright** — full brightness
- **Infinite Jump** — unlimited jumps
- **Speed Slider** — speed multiplier: `0.001x` – `8Bx` ⚡
- **Jump Slider** — jump multiplier: `1x` – `100x`
- **Fly Speed Slider** — fly speed: `0.001x` – `1500x`
- **Speed Method Selector**:
  - `Off`, `BodyVelocity`, `WalkSpeed`, `CFrame`, `Collide`, `Legit`, `Tween`, `Velocity`, `Strength`
- **Fly Mode** — **WASD** control
- **ESP** — player highlights + beams
- **Click Teleport** — teleport on click
- **Tween Move** — smooth movement (LMB to move)
- **Fling** — launch objects/players
- **Noclip** — walk through walls
- **Notifications**, GUI resize, GUI drag

---

## 🛠️ Grabber & Puncher (Physical Interaction Tools)

- **Puncher** — click on any object to launch it away
- **Grabber** — tool that allows you to pick up and hold objects

> 💡 *If these don't activate immediately, try nudging, jumping, or touching the target.*

---

## ⚠️ **Fused** Tab

> Contains various scripts. **Some may be broken or low quality.** Use with caution.

---

## 🕹️ Controls

- **RMB** — AimBot (snap to nearest player)
- **LMB** — Tween Move / Puncher (depending on mode)
- GUI drag — click and drag to move
- GUI resize — via sliders/drag zones
